KSR Definition

Key System Reports are commonly represented as financial data reports. Auditors are required to verify factors that may have an impact on the effectiveness and reliability of the executed report during a SOX audit mission.

When a report is run, a code is generated in the background. As a result, the audited entity has the flexibility to shift this code. As a result, we are required to verify two elements:

  • - UAT (User Acceptance Testing)
  • - Last modification date of code
KSR Types

Furthermore, KSR scope includes three types of reports:

  • -Standard Report
  • -Specific Report
  • -Query with parameters
